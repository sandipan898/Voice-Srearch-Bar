# VoiceSrearch
### A project to make a search bar that can search things on internet and have the feature of voice search

### You can choose your favourite search engines. 
  #### Available search engine options: 
        * Google 
        * Amazone 
        * Youtube 
        * DuckDuckGO

## modules used
1. tkinter 
2. pygame
3. webbrowser
4. Speech Recognition.
